Esc deactivates the current tool.

Shift+click is multiselect. More than one shape can be selected.

While drawing if ctrl is pushed the face picker is disabled

Shift+Delete displays a dialog with all shapes from scene. The user can choose the shape to delete.

Mouse scroll is Zoom in/out

Scroll click and mouse move is Pan

Ctrl+z and Ctrl+y are Undo Redo operations
