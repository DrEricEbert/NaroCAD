While drawing shapes it is possible to enter coordinates also from command line.

There are two types of coordinates:
   - global coordinates, written as: 100,100,100
   - relative coordinates, written as: 100r,100r,0r
This type of coordinate is relative to the previous coordinate inserted from mouse click or command line.

Example:
   first point: 100,100,0
   second point: 100r,100r,0r
The second point will have the global coordinates 200,200,0
