Rectangle function can draw a rectangle

Syntax:

void rectangle(pointt1: integer; point2: integer)

Arguments:

point1: index of the point node representing one corner of the rectangle

point2: index of the point node representing the opposite corner of the rectangle


Sample usage:

s1=Sketch(0,0,5,2,3,5,6,7,5)

EnterSketch(s1)

p1=Point(s1,2,2,5)

p2=Point(s1,15,16,5)

Rectangle(p1,p2)
