Circle function - draws a circle

Syntax - 1:

void circle(x1: real; y1: real, z1: real; radius: real;)

Arguments:

x1, y1, z1: coordinates of the center point

radius: the radius


Syntax - 1:

void circle(point: integer; radius: real;)

Arguments:

point1: index of the point Node representing the center of the circle

radius: the radius


Sample usage:

s1=Sketch(0,0,5,6,3,5,2,3,5) EnterSketch(s1) p1=Point(s1,0,0,5) Circle(p1,20)
