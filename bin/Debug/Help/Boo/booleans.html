Cuts away the shared area of selected objects.

Syntax:

void BooleanSubstract(shapeID1: integer; shapeID2: integer)

Arguments:

shapeID1: first shape ID

shapeID2: second shape ID


Cuts away the shared area of selected objects and creates a single object from the unshared areas.


Sample usage:

s1=Sketch(0,0,0,4,5,0,2,3,0)

EnterSketch(s1)

p1=Point(s1,5,6,0)

c1=Circle(p1,30)

p2=Point(s1,2,2,0)

c2=Circle(p2,10)

e1=Extrude(s1,20)

s2=Sketch(0,0,0,4,5,0,2,3,0)

EnterSketch(s2)

p4=Point(s2,20,20,0)

c4=Circle(p4,5)

e2=Extrude(s2,30)

BooleanSubstract(e1,e2)
