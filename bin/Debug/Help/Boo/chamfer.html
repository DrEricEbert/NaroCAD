Applies a chamfer on 2 selected wire objects.

Syntax:

void Chamfer2D(shape1: integer; shape2:integer; size: real)

Arguments:

shape1: index of the first wire Node

shape2: index of the second wire Node

size: length of the chamfer


Sample usage:

s1=Sketch(0,0,0,5,4,0,2,3,0)

EnterSketch(s1)

p1=Point(s1,2,3,0)

p2=Point(s1,16,19,0)

p3=Point(s1,4,15,0)

l1=Line(p1,p2)

l2=Line(p1,p3)

Chamfer2D(l1,l2,4)
