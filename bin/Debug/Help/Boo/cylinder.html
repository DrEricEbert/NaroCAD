Draws a cylinder from a base circle and a height.

Syntax:

void cylinder(x: real; y: real, z: real; radius: real; height: real; angle: real)

Arguments:

x: x coordinate of the base center point

y: y coordinate of the base center point

z: z coordinate of the base center point

radius: base radius

height: cylinder height

angle: cylinder base arc angle (in degrees)

The orientation is given by the default GpDir


Sample usage:

Cylinder(5,5,5,20,20,120)
