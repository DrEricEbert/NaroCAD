Arc function - draws an arc with the given Center, Start and End

Syntax - 1:

void Arc(x1: real; y1: real, z1: real; x2: real; y2: real; z2: real; x3: real; y3: real; z3: real)

Arguments:

x1, y1, z1: coordinate of the first point representing the center of the arc

x2, y2, z2: coordinate of the second point, representing the start point

x3, y3, z3: coordinate of the second point, representing the end point


Syntax - 2:

void Arc(point1: integer; point2: integer, point3: integer)

Arguments:

point1: index of the point Node representing the center of the arc

point2: index of the point Node representing the start point

point3: index of the point Node representing the end point


Sample usage:

s1=Sketch(0,0,5,6,3,5,2,3,5)

EnterSketch(s1)

p1=Point(s1,0,0,5)

p2=Point(s1,3,0,5)

p3=Point(s1,0,3,5)

Arc(p1,p2,p3)
