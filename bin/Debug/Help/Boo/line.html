Line function can draw a line

Syntax - 1:

void line(x1: real; y1: real, z1: real; x2: real; y2: real; z2: real)

Arguments:

x1, y1, z1: coordinates of the first point

x2, y2, z2: coordinates of the second point


Syntax - 2:

void line(point1: integer; point2: integer)

Arguments:

point1: index of the point Node representing the first point

point2: index of the point Node representing the second point


Sample usage:

s1 = Sketch(0,0,0,2,3,0,5,6,0)

EnterSketch(s1)

p1 = Point(s1,5,5,0)

p2 = Point(s1,10,10,0)

Line(p1,p2)
