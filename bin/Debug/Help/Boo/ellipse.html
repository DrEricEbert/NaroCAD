Ellipse function - draws an ellipse

Syntax - 1:

void ellipse(x1: real; y1: real, z1: real; x2: real; y2: real; z2: real; x3: real; y3: real, z3: real)

Arguments:

x1, y1, z1: coordinates of the center point

x2, y2, z2: coordinates of the point which determines ellipse length

x3, y3, z3: coordinates of the point which determines ellipse width


Syntax - 2:

void ellipse(point1: integer; point2: integer; point3: integer)

Arguments:

point1: index of the point Node representing the center point

point2: index of the point Node representing the point which determines ellipse length

point3: index of the point Node representing the point which determines ellipse width


Sample usage:

s1=Sketch(0,0,0,6,3,0,2,3,0)

EnterSketch(s1)

p1=Point(s1,0,0,0)

p2=Point(s1,8,3,0)

p3=Point(s1,5,6,0)

Ellipse(p1,p2,p3)
