Draw a cone from a base circle and a height.

Syntax:

void cone(x: real; y: real, z: real; base radius: real; upper radius: real; height: real; angle: real)

Arguments:

x: x coordinate of the base center point

y: y coordinate of the base center point

z: z coordinate of the base center point

base radius: base radius

upper radius: upper radius

height: cone height

angle: cone base arc angle in degrees

The orientation is given by the default GpDir


Sample usage:

Cone(0,0,0,30,10,20,360)
