Point function can draw a point in the given sketch

Syntax:

void Point(sketchIndex: integer; x: real; y: real; z: real)

Arguments:

sketchIndex: the index of the sketch node that the point is drawn on

x, y, z: coordinate of the point


Sample usage:

s1 = Sketch(0,0,0,2,3,0,5,6,0)

EnterSketch(s1)

p1 = Point(s1,5,5,0)
